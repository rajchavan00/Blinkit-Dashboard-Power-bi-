Blinkit Sales Performance Dashboard 🛒📊


Description
This project is an interactive Power BI dashboard that tracks and analyzes the sales performance, inventory distribution, and customer ratings for Blinkit (India's Last Minute's App). The goal of this analysis was to uncover actionable business insights regarding outlet growth, item performance, and consumer preferences across various store types and tier locations.

It provides a high-level overview for stakeholders to easily monitor key metrics while allowing deep dives into specific product categories and outlet sizes.

Tech Stack
Business Intelligence: Microsoft Power BI (Data Modeling, DAX, Interactive Dashboards)

Data Cleaning & Transformation: Power Query

Data Source Format: CSV / Excel

Data Source
The raw dataset used for this project was sourced from Kaggle. It contains detailed records of retail sales, item characteristics (like fat content and item type), and outlet attributes (size, location tier, and establishment year).

Key Features & Insights
High-Level KPIs: Quick-glance cards tracking Total Sales ($1.20M), Average Sales ($140), Total Number of Items (9K), and Average Customer Rating (3.92).

Interactive Filtering: A dedicated filter panel allowing users to slice the data globally by Outlet Size, Outlet Location (Tier), and specific Item Types.

Historical Growth Tracking: A time-series line chart illustrating the establishment of outlets from 2012 to 2022, highlighting peak growth years.

Product Category Analysis: Horizontal bar charts breaking down the top-performing item types (with Fruits/Vegetables and Snack Foods leading the pack).

Health & Preference Trends: Donut and stacked bar charts analyzing the distribution of "Low Fat" vs. "Regular" items across the entire network and filtered by specific outlet tiers.

Outlet Performance Matrix: A detailed tabular view comparing different store types (Supermarket Type 1, 2, 3, and Grocery Stores) against key metrics like Total Sales, Average Sales, and Item Visibility.
